# pyPDFTitle
用Python提取PDF文件标题的一个小程序

## 文件介绍
pyPdf-1.13.win32.exe: 配合Python2使用
test.py: 程序主体
foo.txt: 提取PDF标题名的样式

## 使用说明
* 电脑要安装Python2
* 安装pyPdf-1.13.win32.exe，注意需配合Python2使用，Python3存在问题
* 将test.py放在所要获取PDF标题的文件夹内
* 在DOS窗口下cd到该文件夹下
* 执行命令：python -2 test.py
* 打开foo.txt即是你想要的结果
* Note: 无法识别中文
